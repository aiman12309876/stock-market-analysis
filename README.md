# Technical Analysis & Price Action - DecodeLabs Stock Market Project 1

## What It Does
Analyzes stock market data using technical indicators including:
- Support and Resistance levels
- RSI (Relative Strength Index)
- 50-day and 200-day Moving Averages
- Trend identification

## Features
- Fetches live stock data using yfinance
- Calculates key support/resistance levels
- Generates RSI indicator
- Plots price action with indicators
- Provides buy/sell recommendations

## How to Run
1. Install: `pip install pandas numpy matplotlib yfinance`
2. Run: `python stock_analysis.py`
3. Enter a stock symbol (e.g., AAPL, TSLA)
